# Changelog

All notable QSign changes are recorded in this file.

## v0.2 — Document Rendering Engine

- Added PDF opening and page rendering through PyMuPDF.
- Added page navigation, zoom, and bounded rendering cache.
- Added the Flet desktop document viewer.

## v0.1 — Foundation Architecture

- Established the modular service and provider architecture.
- Added domain models, dependency injection boundaries, and logging facade.
- Added the initial Flet desktop shell and unit-test foundation.
